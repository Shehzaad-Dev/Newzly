# 📰 Newzly – Real-Time News App

Newzly is a Flutter-based real-time news app that delivers breaking headlines and categorized news from trusted sources around the world — all packed in a fast, clean UI.

---

## 🚀 Features
- 📍 Top headlines from multiple countries
- 🗂 News categories (Tech, Sports, Business, Health, etc.)
- 🔍 Search functionality
- 🖼 Article preview with image, title, date, source
- 🌐 Open full news articles in web view
- 💡 Light & Dark Mode ready
- 📦 Optimized performance and caching

---

## 🛠️ Tech Stack
- **Flutter** (UI/UX)
- **Dart** (Core language)
- **NewsAPI.org** (Live news source)
- **HTTP Package** (REST API)
- **Provider or GetX** (State Management)
- **Cached Network Image** (Image optimization)
